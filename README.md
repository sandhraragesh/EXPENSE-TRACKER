# Tracker App

A smart tracking application built using Python and Machine Learning.

## Features

- Track user/device/location data
- Predict patterns using ML algorithms
- Data visualization
- CSV dataset support
- Real-time analysis
- Easy-to-use interface

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- VS Code

## Algorithms Used

- Random Forest
- XGBoost
- Decision Tree
- Logistic Regression

## Installation

1. Clone the repository

```bash
git clone https://github.com/yourusername/tracker-app.git
```

2. Open project folder

```bash
cd tracker-app
```

3. Install dependencies

```bash
pip install -r requirements.txt
```

4. Run the app

```bash
python app.py
```

## Dataset

The project uses CSV datasets for training and prediction.
